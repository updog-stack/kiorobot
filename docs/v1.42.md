## 개요
전체현황의 **거래(VAN) 건수**가 고정 샘플값이라 5월까지만 표시되던 문제를 실데이터로 바로잡고, 매출·CMS·VAN 그래프에 **좌측 금액 눈금(Y축)** 을 넣어 막대 높이만으로 대략적인 금액을 읽을 수 있게 했습니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🔴 **전체현황 VAN 건수 실데이터** | 코밴·다우 거래 건수가 **고정 샘플(1~5월)** 이라 5월까지만 나오던 문제 해결. `/api/tr` 실데이터로 **7월(현재월)까지** 자동 표시, 작년(2025) 동기 비교도 실데이터. (KICC는 전용 스크래퍼가 없어 정적 참고값 유지) |
| 🆕 **매출 그래프 금액 눈금(Y축)** | 총매출·CMS·VAN 월별 그래프에 **좌측 금액 눈금 + 가로 격자선**. 데이터 최댓값에 맞춰 4구간 라운드 눈금이 자동 생성되고, **1억 넘으면 '만'→'억' 단위 자동 전환** |
| ⬆️ **월 기준선 정렬** | 총매출 그래프의 '구분 체크박스' 줄만큼 CMS 그래프에도 여백을 넣어, 나란한 두 그래프의 **월·막대 바닥선이 정렬** |

# v1.41 → v1.42 파일별 변경 내역

**수정 3개** (신규 파일 없음).

## A. 수정 파일 (Modified)

### 1. `dashboard/src/lib/tr.ts`
- `trMonthly(tr, key)` 헬퍼 추가 — `/api/tr`의 `series`(코밴/다우 월별)를 특정 VAN의 **올해·작년 12개월 배열 + 데이터 있는 마지막 달**로 변환.

### 2. `dashboard/src/components/Overview.tsx`
- 전체현황 VAN 카드·그래프(daou·kovan·van)를 정적 샘플 → **`/api/tr` 실데이터 라이브**로 연결(코밴·다우, KICC는 정적 폴백·합산엔 데이터 있는 달만 반영).
- `MonthBars`에 **좌측 금액 눈금(Y축) + 가로 격자선**(데이터 기반 라운드 스케일, 만/억 자동) 추가.
- CMS·VAN 그래프(YoYChart)에 총매출 구분 선택줄과 같은 높이 여백(ghost)으로 **월 기준선 정렬**.

### 3. `dashboard/src/App.css`
- 월별 막대 차트 v2 스타일(`.mchart*` — Y축·격자선·막대·월 라벨), 구분 선택줄 높이 고정.
