## 개요
매출·거래 데이터를 더 깊게 보고, CS 대응을 놓치지 않으며, 운영(비번·계정) 관리를 손 안 대고 하도록 자동화했습니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **월별 결제 추이 그래프** | 거래(TR)현황에 **2025년~ 월별 결제 건수(코밴+다우 스택) + 결제 금액(다우)** 그래프. **좌측 상단 년도 필터**로 1년치씩, 2027·2028 자동 누적. 매일 08:00 자동수집·즉시 동기화 |
| 🆕 **다우 결제금액 수집** | 다우데이타 스크래퍼가 '합계 건수'에 더해 **'합계 금액'**도 수집(엑셀). 코밴·다우 스크래퍼 **다년도 백필** 지원 |
| 🆕 **아무도없개 새 상담 슬랙 알림** | 채널톡에 '아무도없개' 상담(이름·태그·콜라인으로 판별)이 오면 **지정한 사람에게 슬랙 DM** 자동 전송(3분 폴링·중복방지·첫 문의 미리보기) |
| 🆕 **VAN 계정 노션 자동로드** | 코밴·다우 **로그인 비번을 노션 계정표에서 자동 로드**. 비번 바뀌면 **노션 한 칸만 수정**(SSH·재시작 불필요, 실패 시 .env 폴백) |
| ⬆️ **CS '오늘 처리' 정확화** | 채널톡 24시간 자동종료로 어제 상담이 오늘 처리로 잡히던 착시 제거 → **오늘 인입(생성) 기준**으로 집계 |
| ⬆️ **꿀팁 상담 수집 한도 제거** | AI 상담검색 수집이 한도(3000)에서 끊기지 않고 **끝까지 자동 반복** 수집(진행상황 표시) |
| 🔧 **매출 동기화 복구** | 노션 매출 DB 재생성으로 바뀐 데이터소스 ID 갱신(장비매출·할부/렌탈) → 전체현황 매출 정상 |

## 설정 (.env 추가/신규)
- `AMUDO_ALERT_SLACK` : 아무도없개 알림 받을 사람의 슬랙 멤버ID(U…). 비우면 알림 비활성.
- VAN 로그인 비번은 이제 노션 **"[다인] VAN/할부/CMS 계정" DB**(라함 통합과 공유)에서 자동 로드 — 비번 변경은 노션에서.
