## 개요
v1.11에서 만든 프롬프트 생성기는 주제를 넣으면 **플랫폼별 완성 콘텐츠(제목·해시태그·본문)** 를 바로 뽑아 주었습니다.
그런데 사용자가 실제로 원한 것은 "완성된 글"이 아니라, **AI 채팅창(ChatGPT·Claude 등)에 붙여넣어 반복적으로 콘텐츠를 만들 수 있는 '지시문(프롬프트)' 그 자체**였습니다.

v1.41에서는 이 메뉴를 **프롬프트(지시문) 생성기**로 개편했습니다. 이제 결과물은 콘텐츠(글·대본)가 아니라 **AI에게 시킬 명령문**이며, 사용자는 그것을 복사해 AI 대화 맨 처음에 붙여넣고 `주제: OO` 형식으로 소재만 던지면 됩니다.

> 참고 골든샘플: `무인매장_시리즈_콘텐츠_제작_프롬프트.md`의 구조(역할→목적→타겟→정보검증→서술원칙→작성규칙→영상 TTS 규칙→시리즈→출력형식→금지사항)를 생성 기준으로 서버 프롬프트에 내장.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🔄 **결과물 전환** | 플랫폼별 '완성 콘텐츠' → **붙여넣어 쓰는 '지시문(프롬프트)'** 생성. 실제 글이 아니라 그 글을 만들도록 시키는 명령문 |
| 🆕 **산출물 선택** | 만들 콘텐츠를 선택: 블로그 게시글 · 유튜브 영상 스크립트 · 인스타그램 · 틱톡·숏폼 · 네이버 블로그(SEO). 선택에 맞춰 작성 규칙 섹션이 달라짐 |
| 🆕 **영상 TTS 규칙** | '유튜브 영상 스크립트'를 고르면, 그 글이 그대로 AI 영상 툴에서 TTS로 읽히도록 하는 규칙(문장 60자 이내·이모지/표/특수문자 금지·URL은 맨 아래 등)이 프롬프트에 자동 포함 |
| 🆕 **입력 항목** | 분야/주제 + 산출물(다중) + 타겟 독자 + 역할·말투(페르소나) + 추가 지침 + 모델 |
| ⬆️ **결과 UI** | 완성 프롬프트를 큰 텍스트영역 + '프롬프트 복사' 버튼으로 제공, 사용법 안내와 비용(원) 표시 |

## 사용법
1. 분야/주제와 만들 콘텐츠를 고르고(선택적으로 타겟·말투·추가지침) 생성한다.
2. 나온 **지시문 프롬프트를 복사**한다.
3. ChatGPT·Claude 대화 **맨 처음에 붙여넣고**, 이후 `주제: (소재)` 형식으로 던진다.
4. 같은 대화에서 주제만 바꿔가며 반복 사용한다.

> 프롬프트가 "법령·가격·지원금 등 시점에 따라 달라지는 정보는 최신으로 확인하라"고 지시하므로, 실제 사용 시 **웹 검색(브라우징) 기능이 켜진 모드**에서 쓰면 정확도가 올라간다.

## 설정
- 생성에는 `ANTHROPIC_API_KEY`(.env)를 사용합니다(블로그 검사기와 동일 키·모델 라인업 재사용: Sonnet 4.6 / Haiku 4.5 / Opus 4.8).

# v1.40 → v1.41 파일별 변경 내역

**수정 3개** (신규 파일 없음).

## A. 수정 파일 (Modified)

### 1. `dashboard/server/notion-sales-bff.mjs`
- 프롬프트 생성기 엔드포인트를 **`/api/content-generate`(완성 콘텐츠) → `/api/prompt-generate`(지시문 생성)** 로 교체.
- `CONTENT_PLATFORMS`(플랫폼별 콘텐츠 규칙) → **`PROMPT_DELIVERABLES`**(산출물별 작성 규칙: blog·video·instagram·tiktok·naver)로 대체.
- 시스템 프롬프트를 '프롬프트 엔지니어' 역할로 재작성 — 콘텐츠가 아니라 **완성 지시문**을 출력하게 하고, 11개 섹션(사용법·역할·목적·타겟·정보검증·서술원칙·작성규칙·(영상 시)TTS 규칙·시리즈·출력형식·금지사항) 구조를 강제. 순수 텍스트 프롬프트 반환 + 토큰 기반 비용(원) 계산.

### 2. `dashboard/src/lib/promptgen.ts`
- 타입을 `PromptResult`(prompt 문자열)·`GeneratePromptInput`(topic·deliverables·audience·persona·extra)로 개편.
- `DELIVERABLES`(산출물 목록)·`PROMPT_MODELS` 추가, `generateContent()` → **`generatePrompt()`**(`/api/prompt-generate` 호출)로 교체.

### 3. `dashboard/src/components/PromptGen.tsx`
- 입력 폼을 분야/주제 + 산출물 다중선택 + 타겟·역할말투·추가지침 + 모델로 개편.
- 결과를 플랫폼별 카드 → **완성 프롬프트 1건(큰 textarea + '프롬프트 복사' + 사용법 안내)** 으로 변경.
