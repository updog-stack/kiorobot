## 개요
외부 마케팅 채널(당근 광고·유튜브·네이버 블로그)의 현황을 한 곳에서 보는 **'마케팅 현황' 메뉴**를 신설했습니다. 또 VAN 단말기가 몇 대 개통돼 실제로 쓰이는지(휴면 여부)를 전체현황에서 봅니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **마케팅 현황 (새 메뉴)** | 전체보기 / 🥕당근마켓 광고 / 📺유튜브 / 📝블로그 탭. 유튜브는 전체현황에서 이 메뉴로 이동 |
| 🥕 **당근마켓 광고** | 광고캐시·노출·클릭·CTR·지출 + 광고별(디스플레이·검색). **상주 브라우저 세션**으로 30분마다 수집(`당근광고수집.bat`). 당근은 SMS/SSO 로그인이라 무인 자동수집 불가 → 세션 유지 방식 |
| 📝 **네이버 블로그 조회수** | 일별 조회수(오늘·7일·30일 + 최근 14일 추이 그래프, 좌측 눈금). 네이버 세션 재사용 수집(`네이버블로그수집.bat`), 만료 시 `네이버블로그로그인.bat` |
| 🆕 **단말기 사용현황 (전체현황)** | VAN별 개통/사용(최근 7일)/휴면(7일+ 미결제). **코밴 정밀 7일**(단말기별 일자 데이터), **다우 일별누적**(월 데이터라 7일 워밍업 후 정밀) |

## 설정 (.env / 실행)
- `NAVER_BLOG_ID`(기본 dain_inc), `DAANGN_ADVERTISER_ID`(기본 3794527) — .env로 변경 가능.
- 당근·네이버는 **로그인 인증(헤드리스 차단·SMS/SSO/세션 revoke)** 때문에 **서버(무인) 자동수집이 불가** → 로컬 PC에서 `.bat`(창 뜨는 방식)으로 수집. 세션 만료 시 재로그인.
- 단말기 수집(코밴·다우)은 헤드리스 가능 → 8시 자동수집(COLLECT_SCRIPTS)에 포함.
