## 개요
그동안 "업무 / 할 일" 탭은 자리만 잡혀 있었습니다. v1.3에서 이를 **실제 기능 2개**로 채웠습니다.
회사 노션의 **[업무 DB]** 를 그대로 끌어와, 팀 전체 업무 현황을 한눈에 보고(업무현황) 하루를 자동으로 정리(업무일지)합니다.

핵심 2가지:
1. **업무현황** — 노션 업무 DB를 실시간(5분 갱신)으로 모아 **상태·담당자별 집계 + 정체 업무 표시 + 필터/검색**.
2. **업무일지** — 매일 **오후 6시 자동 생성**되는 하루치 일지(담당자별 완료/진행/보류/정체 + **AI 요약 코멘트**), 날짜별 조회·수동 생성.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **업무현황** | 노션 [업무 DB] → 진행중·대기·보류·오늘완료·**정체** KPI(클릭 필터), **담당자별 현황표**, 업무 목록(담당자·상태·검색 필터). 5분 자동 갱신 |
| 🆕 **업무일지** | 매일 **18:00 자동 생성**(서버 꺼져 있었으면 다음 구동 시 보완) + "지금 생성" 수동. 담당자별 완료/진행/보류/정체 정리 + **Claude AI 요약 코멘트**. 날짜별 조회 |
| ⬆️ **내비게이션** | 빈 "업무 / 할 일" 탭 → **업무현황(✅)·업무일지(📝)** 두 탭으로 대체 |

## 설정
- 노션 통합(integration)이 **[업무 DB]** 데이터소스에 접근 권한이 있어야 합니다(기존 `NOTION_TOKEN` 사용).
- 업무 DB 데이터소스 ID는 코드에 기본값이 있고, 바꾸려면 `dashboard/.env` 의 `NOTION_TASKS_DS_ID` 로 덮어쓸 수 있습니다.
- AI 요약 코멘트는 `ANTHROPIC_API_KEY` 를 사용합니다(없으면 코멘트만 생략, 일지 생성은 정상).

> 업무 DB 속성 매핑: 업무명·상태·담당자T/담당자명·담당자직책·우선순위·연관부서·업무내용·업무일자·진행시작일·완료일·마지막상태변경일·정체플래그·휴지통. (휴지통 항목은 제외)
