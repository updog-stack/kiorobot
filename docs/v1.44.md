## 개요
좌측 메뉴가 많아져 보기 힘든 문제를 **탭 통합**으로 정리하고, 그동안 "금액 미제공"이던 **코밴 결제금액**을 포털의 금액구간집계에서 수집해 거래(TR) 현황 그래프에 넣었습니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| ✅ **업무현황 (통합)** | 기존 **CS현황·업무현황·업무일지** 3개 메뉴를 **한 메뉴 + 탭**으로 통합. 대표 탭=업무현황 |
| 🪄 **AI 업무지원 (통합)** | 기존 **꿀팁게시판·블로그 검사기·프롬프트 생성기** 3개를 **한 메뉴 + 탭**으로 통합 |
| 💳 **코밴 결제금액 (신규 실데이터)** | 거래(TR) 현황 → 월별 결제 추이의 금액 그래프를 **코밴+다우 스택**으로. '결제 금액(코밴)' KPI 카드 추가 |

## 코밴 금액의 한계 (중요 — 화면에도 "근사" 표기)
코밴 포털(catECA)은 다우처럼 정확한 총액을 주지 않아, **가맹점 금액구간집계**(`fixedrate.aspx`)의 합계를 사용합니다. 따라서:
- **카드(신용+체크)만** — 현금 제외
- **100만원 초과 건은 100만원으로 절삭** 표기 — 큰 결제일수록 실제보다 낮게 잡힘
- **1천원 이하 제외**
- **1년 이전 월은 조회 불가** — 2025년 상반기 금액은 공란
- **당월은 '어제까지'** 집계(당일 미정산분이 있으면 포털이 범위 전체를 0으로 반환하는 특성 회피)

즉 코밴 금액은 **실제보다 약간 낮은 근사치**이며, 건수(정확)와 달리 참고용입니다. 다우데이타 금액은 종전대로 정확한 실데이터입니다.

## 수집된 코밴 월별 금액(예: 2026년, 억원)
1월 22.99 · 2월 20.65 · 3월 19.12 · 4월 19.04 · 5월 21.26 · 6월 19.24 · 7월 4.77(어제까지) — 2025-07~12도 함께 백필.
