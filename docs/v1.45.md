## 개요
흩어져 있던 매출 데이터를 한 곳에 모은 **'매출현황' 메뉴**를 신설하고, 코밴 결제금액을 작년 미수집분까지 예측해 전년 대비를 완성했습니다. 단말기·가맹점 미사용 매장을 클릭해 상호/사업자번호로 확인할 수 있고, '데이터 동기화'는 현재 보는 화면만 빠르게 갱신하도록 바꿨습니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **매출현황 (새 메뉴)** | 총매출(장비·라이선스·기타)·VAN 결제금액(코밴+다우)·CMS 수납액을 **중복 없이 한 메뉴**에. 기존 컴포넌트 재사용(전체현황·거래현황은 그대로) |
| 💳 **코밴 결제금액 예측·전년비** | 포털 1년 제한으로 미수집된 작년 월은 **건수 × 평균단가로 예측**해 채우고 전년 동기 대비·월평균 표시(예측 월은 옅은 색·표기) |
| 📊 **CMS 매출 강화** | 월평균·이번달 **전년 대비 배지**, **역대 최고/최저**, 그래프 좌측 금액 눈금 |
| 🏪 **운영 가맹점 수** | 사업자번호 기준 distinct — 코밴·다우 **중복(양쪽 사용) 제거**, KICC 제외. 개통/운영(7일)/미사용 |
| 🔎 **미사용 매장 드릴다운** | 단말기·가맹점 카드의 **미사용 클릭 → 상호·사업자번호 명단**. 다우는 **7일 미결제 기준으로 통일**(코밴과 동일, 카드=명단 일치) |
| 🔄 **데이터 동기화 = 현재 화면만** | 수동 버튼은 **보고 있는 페이지에 필요한 스크래퍼만** 실행 후 그 화면 자동 갱신. 전체 수집은 매일 08:00 자동 |
