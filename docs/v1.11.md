## 개요
마케팅 콘텐츠를 매번 직접 쓰던 것을, **주제/키워드 한 줄**로 여러 플랫폼용 완성 콘텐츠를 한 번에 만들도록 자동화했습니다.

프롬프트 생성기는 주제를 넣으면 **유튜브·인스타그램·틱톡·네이버 블로그** 각각에 **바로 복사해 올릴 수 있는** 제목·설명(본문)·해시태그·태그·노출 팁을 생성합니다. 각 플랫폼의 노출(검색·추천) 알고리즘 기준에 맞춰 최적화되며, 결과는 플랫폼별 카드로 정리되어 복사 버튼으로 즉시 활용할 수 있습니다.

핵심:
1. **완성 콘텐츠 즉시 생성** — 프롬프트가 아니라 그대로 게시할 결과물(제목·본문·해시태그)을 출력.
2. **플랫폼별 노출 최적화** — 유튜브(검색+추천), 인스타/틱톡(해시태그 도달·추천), 네이버 블로그(검색 SEO) 기준을 서버 프롬프트에 내장.
3. **'노출이 쉽고 잘 되는' 전략** — 경쟁 극심한 대형 키워드 대신, 검색 수요는 있으면서 경쟁이 낮은 **롱테일·틈새 키워드**를 제목·첫 문장·해시태그 앞쪽에 우선 배치.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **프롬프트 생성기** | 주제/키워드 + 플랫폼(다중선택) + 톤 + 모델 → 플랫폼별 **제목 후보·본문/설명·해시태그·검색태그·노출팁** 생성. 블로그 검사기와 동일한 **모델 선택 + 비용(원) 표시** |
| 🆕 **4개 플랫폼** | 유튜브(검색태그 10~15) · 인스타그램(해시태그 15~25) · 틱톡·스레드(숏폼 후킹) · 네이버 블로그(SEO 본문 초안 + 태그) |
| ⬆️ **노출 최적화 전략** | 롱테일·틈새 키워드 우선, 대형+틈새 해시태그 혼합, 검색·클릭 표현을 앞쪽 배치 |
| ⬆️ **내비게이션** | '블로그 검사기'와 '꿀팁게시판' 사이에 **프롬프트 생성기(🪄)** 추가 → 메뉴 10개 → 11개 |

## 설정
- 생성에는 `ANTHROPIC_API_KEY`(.env)를 사용합니다(블로그 검사기와 동일 키·모델 라인업 재사용: Sonnet 4.6 / Haiku 4.5 / Opus 4.8).
- 별도 외부 연동·DB는 필요 없습니다(주제 텍스트만 입력).

> 플랫폼별 규칙은 서버의 `CONTENT_PLATFORMS`(youtube·instagram·tiktok·naver)에 정의되어 있어, 새 플랫폼 추가·기준 조정이 쉽습니다.

# v1.10 → v1.11 파일별 변경 내역

**신규 2개** + **수정 2개**.

## A. 신규 파일 (New)

### 1. `dashboard/src/lib/promptgen.ts` (신규)
- 프롬프트 생성기 데이터 레이어: `PlatformContent`·`ContentResult` 타입, 선택용 목록(`PLATFORMS`·`TONES`·`CONTENT_MODELS`), `generateContent()`(`/api/content-generate` 호출, BFF 미실행 시 친절 안내).

### 2. `dashboard/src/components/PromptGen.tsx` (신규)
- **'프롬프트 생성기' 탭** 전체. 주제 입력, 플랫폼 다중선택(칩 버튼·초기 미선택), 톤·모델 선택, 추가 정보. 결과는 플랫폼별 카드(제목 후보·본문/설명 textarea·해시태그·검색태그·노출팁)로, 각 항목·전체 복사 버튼과 비용(원) 표시 제공.

## B. 수정 파일 (Modified)

### 3. `dashboard/server/notion-sales-bff.mjs`
- **`POST /api/content-generate`** 신규 — `CONTENT_PLATFORMS`(유튜브·인스타·틱톡·네이버 노출 기준)·`CONTENT_TONES` 정의, 요청 플랫폼만 필터링해 시스템 프롬프트 구성, Claude 호출 → 순수 JSON 파싱 → 플랫폼 순서대로 정리, 토큰 기반 비용(원) 계산. 모델·가격표는 기존 블로그 검사기(`BLOG_MODELS`)와 공유.
- 시스템 프롬프트에 **'노출이 쉽고 잘 되는' 최우선 전략**(롱테일·틈새 키워드 우선, 검색·클릭 표현 앞쪽 배치) 반영.

### 4. `dashboard/src/App.tsx`
- `PromptGen` import, 내비 항목 `프롬프트 생성기`(🪄, blog↔playbooks 사이) 추가, `active === "promptgen"` 렌더 연결.
