# v0.1 — 최초 프로토타입

> 다인아이앤씨 통합 대시보드의 첫 버전. 핵심 운영 기능을 갖춘 **프로토타입**.
> ← [메인 README로 돌아가기](../README.md)

---

## 개요
회사의 여러 시스템(노션·코밴·다우데이타·채널톡·국세청)에 흩어진 데이터를 한 화면에서 보는 대시보드의 **기반**을 구축한 버전입니다.

## 구현된 기능

| 메뉴 | 구현 내용 | 데이터 출처 |
|------|-----------|-------------|
| **경영 지표** | 금일/월간/월평균/년간 매출, 전년 동기 대비 성장률, 월별 매출 차트(올해 vs 작년) | 노션 매출 DB(장비매출·할부/렌탈) |
| **거래(TR) 현황** | VAN사별(코밴·다우데이타) 월별 거래 건수 + 합산, 추이 차트, 수동 동기화 | 코밴 CATECA·다우데이타 DDWM 자동수집 |
| **무실적 가맹점** | 거래 없는 가맹점 목록(매장명·사업자번호·작년매출·연락처), 국세청 폐업여부 자동 조회, 검색 | 코밴·다우데이타 + 국세청 API |
| **CS 현황** | 담당자별 실시간 상태(채널톡), 일일 처리/대기, 상담 매장 목록, 전화·채팅 인입 히트맵 | 채널톡 Open API |
| **꿀팁게시판** | 상담 가이드 Q&A(의사결정 트리), Claude AI 응대 초안 생성 | 수동 작성 + 채널톡 이력 + Claude AI |

> `전체 현황`·`일정`·`업무/할 일` 탭은 자리(placeholder)만 존재.

## 주요 구성 파일 (v0.1 시점)
- `dashboard/src/App.tsx` — 탭 구성·라우팅
- `dashboard/src/components/` — `SalesMetrics`, `TrMetrics`, `InactiveStores`, `CsStatus`, `CallHeatmap`, `Playbooks` 등
- `dashboard/src/lib/` — `sales`, `metrics`, `tr`, `inactive`, `cs`, `calls`, `playbooks`, `format`
- `dashboard/server/notion-sales-bff.mjs` — 중계 서버(BFF), 모든 `/api/*` 처리
- `dashboard/server/*-scraper.mjs` — 코밴·다우데이타 자동수집 스크래퍼

## 실행 방법
```bash
cd dashboard
npm install
npm run dev                          # 화면 → http://localhost:5173
node server/notion-sales-bff.mjs     # 중계 서버 → http://localhost:8787
```
