## 개요
경영지표를 **가짜값(샘플) 없이 실데이터**로 재설계하고, 갤럭시 폴드·모바일에서 쓸 수 있도록 **반응형 레이아웃**을 넣었습니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 📈 **경영지표 재설계** | 핵심 재무 3종(**총매출·CMS 수납·VAN 결제금액**)을 라이브 API로 연동 — 다른 메뉴와 숫자 일치. 성과점수(작년=100점)·연말예상·월별 동월대비·원포인트 프레임워크는 유지 |
| 🏪 **사업 건전성 섹션(신규)** | 운영 가맹점 수·단말기 가동률·미사용률(코밴+다우 실데이터, 현재 상태) |
| 🧹 **가짜값 지표 제거** | CS 처리·콘텐츠 조회수·신규 사업자(전부 임의값)를 경영지표에서 제거 |
| 📱 **모바일·폴드 반응형** | 좁은 화면(≤900px)에서 사이드바를 **햄버거 드로어**로, 본문 1열, 헤더 압축, 표 가로 스크롤. ≤480px 추가 조정 |

## 경영지표의 의미 정리(설계 근거)
- **경영지표 = 대표가 회사 성과를 판단하는 핵심 숫자**. 활동량(CS·콘텐츠)이나 가짜값이 아니라 **재무 성과·사업 건전성**이어야 함.
- 총매출·CMS는 **다인 실매출**, VAN 결제금액은 **가맹점 거래대금(수수료 수익의 기반)** — 배너에 구분 명시.
