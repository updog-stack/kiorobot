## 개요
전체현황·거래현황의 매출 지표가 예시값을 벗어나 **실제 업체·노션 데이터**로 바뀌었습니다. CMS 매출은 효성CMS 실제 수납액, 매출은 노션 장비매출 DB의 실데이터(장비·라이선스·기타)로 자동 연동되고, 시트→노션 동기화의 금액·상태 정확도를 크게 높였습니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **CMS 매출 = 효성CMS 실데이터** | 전체현황 'CMS 매출' 카드·월별그래프 + 거래현황 CMS 섹션이 **효성CMS 실제 월별 수납(완납)액**. 스크래퍼가 통계 API에서 수집, **이메일 2차인증 자동처리**(다른 IP/24h 시 Gmail 코드 자동 입력) |
| 🆕 **총매출 = 노션 실데이터 + 구분 체크박스** | 전체현황 '장비 매출'을 **'총 매출'(장비·라이선스·기타)** 로 전환. 2026년은 노션 장비매출 DB 라이브(데이터 있는 달까지, 7월 자동 반영), 2025년은 시트 실데이터. **그래프에 구분 체크박스**(장비/라이선스/기타)로 올해·작년 동시 토글 |
| ⬆️ **월별 결제추이 값 표시** | 거래현황 막대에 **값 라벨(건수·금액)** + 상단 카드에 **전년 동기간 값·증감률** |
| ⬆️ **아무도없개 알림 봇 표시이름** | 슬랙 DM 발신 봇 표시이름을 **'아무도없개_CS기록'** 으로(chat:write.customize) |
| 🔧 **전체현황 정리** | 샘플 카드(신규 카드가맹·라이센스·이전설치·콘텐츠 발행량·사업자 현황) 제거로 실데이터 위주 화면 |
| 🔧 **노션 매출 동기화 정확도** | 라이선스·기타 **금액 정상 반영**(0원 문제 해결), **별도가(일시불 VAT별도)·총액** 채움, **수금여부·OK사인·효성CMS** 시트값 동기화, 장비내용 올바른 칸, 총액을 **숫자(₩)로 통일**·반올림·중복 칸 정리 |

## 설정 (.env / 노션)
- 효성CMS 스크래퍼는 기존 이메일 2차인증 인프라(`GMAIL_USER`·`GMAIL_APP_PASSWORD`) 재사용.
- 효성CMS 로그인 계정은 노션 **"[다인] VAN/할부/CMS 계정" DB**(서비스명 "효성CMS")에서 자동 로드 — 비번 변경은 노션에서.

# v1.30 → v1.40 파일별 변경 내역

## A. 신규
- `dashboard/server/hyosung-cms-scraper.mjs` — 효성CMS 로그인(+이메일 2차인증) → 통계 API 월별 수납액 수집(`server/data/cms-hyosung.json`).
- `dashboard/src/lib/cms.ts` — `/api/cms` 페치.

## B. 서버(`notion-sales-bff.mjs`)
- **CMS 매출**: `buildCms`·`/api/cms`·`/api/cms/sync`, 8시 자동수집(COLLECT_SCRIPTS)에 효성CMS 포함.
- **총매출**: `buildSalesMonthly`·`/api/sales-monthly`(장비매출 DB 구분별 월별 합산), 금액은 **총액(VAT 포함, 숫자)** 우선 읽기·반올림.

## C. 프론트
- `dashboard/src/lib/overview.ts` — `총매출`(장비·라이선스·기타), `SALES_2025`(3분류 실데이터), 샘플 카드·지표 제거.
- `dashboard/src/components/Overview.tsx` — CMS·총매출 라이브 연동, **총매출 구분 체크박스(TotalSalesChart)**, 샘플 카드 제거.
- `dashboard/src/components/TrMetrics.tsx` — CMS 실데이터, 결제추이 값 라벨·전년 동기간 대비.
- `dashboard/src/App.css` — 구분 체크박스 스타일.

## D. 노션 동기화(구글 Apps Script — 저장소 외)
- 시트 매출액(총액/별도가) → 노션 **총액(VAT 포함)·일시불(VAT 별도)** 정확 매핑, **수금여부·OK사인·효성CMS** status 동기화, 장비내용 매핑 교정, 금액 반올림.
- 노션 장비매출 DB: `총액(VAT 포함)` 수식→숫자(₩) 전환, 임시 `매출액(시트)` 칸 정리.
