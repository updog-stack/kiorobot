# v1.0 — 전체현황·일정·경영지표 확장

> v0.1 기반 위에 **전체 현황 대시보드 · 구글캘린더 일정 · 경영지표 점수화**를 추가한 버전.
> ← [메인 README로 돌아가기](../README.md) · 이전 버전: [v0.1](v0.1.md)

---

## 개요
v0.1이 "데이터를 한 화면에 모으는 기반"이었다면, v1.0은 그 데이터를 **요약·분석·평가**하고
**일정까지 양방향으로 관리**하는 기능을 더한 버전입니다.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **전체 현황** | 핵심 운영지표(장비·CMS 매출, VAN 건수, CS, 콘텐츠, 신규/폐업 사업자)를 한 화면에 요약 |
| 🆕 **일정** | 구글캘린더 월간 캘린더 — 월 이동·상세 보기·**생성/수정/삭제 양방향 동기화(45초 폴링)**·다중 캘린더 |
| ⬆️ **경영 지표** | 월별·작년 **동월** 대비 평가 + **연말 예상(작년 계절성)** + **작년=100점 성과 점수 + 원포인트 처방** |
| ⬆️ **거래(TR) 현황** | **KICC** VAN(구글시트 참고값) + **CMS 매출** 섹션 추가 |
| 🛠 **배포 편의** | `실행.bat`(윈도우 원클릭), 다른 PC 실행·구글캘린더 설정 가이드 |

## 설정 (신규)
- 구글캘린더 연동: `dashboard/.env` 의 `GOOGLE_CALENDAR_ICS_URL`(읽기) 또는 `GOOGLE_SA_*`/`GOOGLE_CALENDARS`(양방향).
  - 자세히: [`dashboard/구글캘린더_연동_설정.md`](../dashboard/구글캘린더_연동_설정.md)
- 다른 PC에서 실행: [`dashboard/다른PC에서_실행하기.md`](../dashboard/다른PC에서_실행하기.md)

# v0.1 → v1.0 파일별 변경 내역 (전체)

v0.1 대비 바뀐 모든 파일을 하나하나 정리했습니다. (총 **19개 파일, +3,453 / −36 줄**)

## A. 신규 파일 (New)

### 1. `dashboard/src/components/Overview.tsx` (신규, 270줄)
- **'전체 현황' 탭** 전체 구현. 핵심 운영지표를 한 화면에 요약.
- 상단 KPI 요약 카드 6종(장비매출·CMS·VAN건수·CS·콘텐츠 조회수·신규사업자 순증) + 작년 대비 증감 배지.
- 섹션 구성: ① 매출(장비·CMS 월별 올해 vs 작년 차트 + 신규카드가맹/라이센스/이전설치 구성) ② 거래(VAN) 3사 카드 + 합산 차트 ③ 운영(CS·콘텐츠) ④ 사업자(신규 vs 폐업 월별).
- 내부 `MonthBars` 막대 차트 컴포넌트, 샘플 데이터 표시 태그.

### 2. `dashboard/src/components/Schedule.tsx` (신규, 455줄)
- **'일정' 탭** 전체. 구글캘린더형 **월간 캘린더 UI**.
- 6주 그리드, 이전/다음 달 이동, '오늘' 버튼, **새로고침 버튼**, **45초 자동 폴링**.
- 일정 클릭 → **상세 모달**(일시·캘린더·장소·세부내용·주최자·구글 링크).
- **생성/수정/삭제 폼**(`EventForm`): 캘린더 선택, 종일 토글, 날짜/시간, 장소, 세부내용. 빈 날짜 클릭 시 그 날짜로 생성.
- 다중 캘린더 색상 범례, 미설정 시 설정 안내 카드, 읽기전용/편집가능 분기.

### 3. `dashboard/src/components/ManagementMetrics.tsx` (신규, 381줄)
- **'경영 지표' 탭** 전체 (기존 `SalesMetrics`를 대체).
- **성과 점수판**: 작년 동기=100점 기준 **종합 점수** + 지표별 점수 카드(등급·점수막대·연말예상 점수) + **원포인트 처방**(작년 넘기 위한 월평균 목표).
- 핵심 요약(YTD·YoY·연말예상), 연말 전망(작년 실적 vs 올해 예상 막대 비교).
- 지표별 상세 블록: 올해 vs 작년 차트 + **월별 동월대비 표** + **자동 인사이트 문장** + 원포인트.

### 4. `dashboard/src/lib/overview.ts` (신규, 307줄)
- 전체현황·경영지표 **공용 데이터셋**: 장비매출·CMS(스프레드시트 예시값), VAN 3사(DAOU/KOVAN/KICC), CS·콘텐츠·매출구성·사업자(샘플).
