## 개요
현장에서 매번 손으로 확인·입력하던 것들을 **자동화**하고, 업무 현황·일지를 **한 화면에서 바로 파악·공유**할 수 있게 고도화했습니다.

핵심:
1. **전체 현황** — 유튜브 채널지표를 카드로 추가하고, VAN 데이터 수집을 버튼 한 번(+매일 8시 자동)으로.
2. **업무현황** — 노션 협업 관계를 그래프로 시각화하고, 내근/외근·업무부하를 한눈에.
3. **업무일지** — 매일 저녁 '데일리 브리핑'을 자동 생성해 캘린더로 누적하고, PDF로 저장해 대표님 슬랙으로 바로 전송.
4. **채널톡 일일 리포트** — 전화·채널톡·카카오 인입을 **전부 자동집계**(더 이상 수동 입력 없음).

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **유튜브 채널지표** | 전체현황에 구독자·총조회수·영상수·최근 영상 카드(YouTube Data API v3, 10분 캐시) |
| 🆕 **데이터 동기화** | 전체현황 '데이터 동기화' 버튼 + **매일 08:00 자동수집**(코밴·다우데이타 TR/무실적 스크래퍼) |
| ⬆️ **업무현황** | 노션 협업(협업자·요청자) **관계 그래프**, 담당자 **내근/외근 직접 변경**, **업무부하 4단계(여유·보통·바쁨·과부하) 자동 판정**, **이름 클릭 → 담당 업무 내용 열람** |
| ⬆️ **업무일지** | 매일 **19:00 자동 생성** '데일리 브리핑' 신문형 양식(세줄요약·수치·하루의 흐름·구성원별·성과/진전/걸림돌·내일), **캘린더 일자별 누적**, **매월 1일 월간 종합**, 1920×1080 한 화면 최적화 |
| 🆕 **PDF 저장·슬랙 전송** | 업무일지 화면을 **PDF로 저장**(화면 그대로/문서형) + **대표님 슬랙으로 전송** |
| 🆕 **CS 일일 리포트 자동집계** | 서비스별(전화·채널톡·카카오) 인입 비율 + 담당자별 상담 수를 **전부 채널톡 Open API로 자동 계산**(수동 입력 제거) |

## 설정 (.env 추가 항목)
- `YOUTUBE_API_KEY` · `YOUTUBE_CHANNEL` — 유튜브 채널지표(서비스계정 아님, API 키).
- `SLACK_BOT_TOKEN` · `SLACK_CEO_CHANNEL` — 업무일지 PDF 대표님 전송(봇 스코프: `files:write`·`chat:write`·`im:write`).
- 채널톡 자동집계·업무일지 AI 요약은 기존 `CHANNELTALK_ACCESS_KEY/SECRET`·`ANTHROPIC_API_KEY` 재사용.
