## 개요
사내에서 쓰던 **파이썬 데스크톱 이미지 리사이저**(`C:\Cording\ImageResizer` — Pillow/OpenCV 기반)를 대시보드 안의 메뉴로 이식했습니다.
별도 프로그램을 띄우지 않고, 대시보드에서 바로 사진 크기를 줄이거나 포맷을 바꿀 수 있습니다.

핵심:
1. **완전 브라우저 처리(서버 업로드 없음)** — 사진이 회사 서버로 올라가지 않아 빠르고 안전합니다.
2. **원본 프로그램 로직 이식** — 고품질 다운스케일(단계적 반감+언샤프), 단계적 업스케일, 비율이 다르면 중앙 크롭(cover).
3. **AI 업스케일(선택)** — 확대 화질을 높이려면 브라우저 AI(UpscalerJS/TensorFlow.js·ESRGAN)로 처리. 기존 파이썬의 ESPCN/EDSR(OpenCV)을 브라우저 실행 모델로 대체.

## 동작·성능 메모
- **AI 업스케일**은 첫 사용 시 모델을 내려받아 처음 한 장은 느리고, 이후엔 빨라집니다. 브라우저 GPU(WebGL)를 사용하며 큰 이미지는 타일(patch)로 자동 분할합니다.
- TensorFlow.js(약 1MB)는 **AI를 켠 순간에만 지연 로딩**되어 별도 청크로 분리됩니다 — 기본 앱 로딩에는 영향이 없습니다.
- 축소·크롭에는 AI 설정이 영향을 주지 않습니다(확대일 때만 적용).

## 설정
- 추가 API 키나 서버 연동이 필요 없습니다(전부 클라이언트에서 처리).
- 의존성 추가: `upscaler`, `@tensorflow/tfjs`, `@upscalerjs/esrgan-slim`.

- 브라우저(Canvas) 이미지 처리 엔진. 목표 크기 계산(`calcTargetSize`), 고품질 다운스케일(단계적 반감+언샤프), 단계적 업스케일, 중앙 크롭(cover), JPG 배경 흰색 처리, Blob 변환. 견고한 디코딩(`loadSource` — `<img>` 우선, 캔버스 폴백, 비이미지 파일 안내, 원본 미리보기 URL).
