# v1.2 — 로그인(접근제어)·블로그 검사기·AI 상담검색·꿀팁 고도화

> v1.0 기반 위에 **외부 서버 오픈을 위한 로그인 + 블로그 검사기 + AI 상담검색(기간 수집) + 꿀팁게시판 고도화**를 더한 버전.
> ← [메인 README로 돌아가기](../README.md) · 이전 버전: [v1.0](v1.0.md) · [v0.1](v0.1.md)

## 개요
v1.0이 "데이터를 모아 요약·분석·일정관리"하는 단계였다면, v1.2는 그 대시보드를
**인터넷(외부 서버)에 안전하게 열고**, 실무에서 바로 쓰는 **AI 보조 도구**(블로그 검사·상담 검색·응대 가이드)를 강화한 버전입니다.

핵심 4가지:
1. **로그인(접근제어)** — 외부 서버에 열어도 비밀번호 없이는 화면·데이터 모두 차단.
2. **블로그 검사기** — 네이버 블로그 글을 Claude가 SEO 관점으로 분석·교정안 제시.
3. **AI 상담 검색** — 과거 채널톡 상담을 **기간 지정으로 수집(중복 제외)** 하고, 질문하면 원포인트 답변.
4. **꿀팁게시판 고도화** — 응대 가이드에 사진·유튜브 첨부, 채널톡 이력 기반 AI 플레이북 생성.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🔐 **로그인/보안** | 앱 내장 로그인(공유 비밀번호 1개) — 로그인 화면 → **세션 쿠키(httpOnly·7일)** → `/api/*` 전체 보호. 로그인 무차별 대입 제한(IP당 15분 10회), 기본 보안 헤더, 전면개방 CORS 제거. 의존성 추가 없이 Node 내장 `crypto` 사용 |
| 🆕 **블로그 검사기** | 네이버 블로그 제목·본문(+이미지)을 Claude로 분석 — **종합 점수·핵심 키워드·문제점/강점·제목 대안·개선 본문**, 키오스크 모델 문맥 반영, **토큰·비용(원) 표시** |
| 🆕 **AI 상담 검색** | 채널톡 종료 상담을 인덱스로 모아 **글자 2-gram + 단어 점수**로 후보 추림 → Claude가 **원포인트 답변·조치 단계·근거 기록** 정리 |
| 🆕 **기간 수집(증분)** | 상담 수집을 **시작일~종료일로 지정**, **이미 모은 상담은 자동 건너뜀(중복 제외)** 후 병합 — 채널톡 호출·시간 절약 |
| ⬆️ **꿀팁게시판** | 기존 의사결정 트리에 **미디어 첨부(사진 업로드·유튜브·텍스트)**, **AI 플레이북 생성 고도화**(분석 기간·최소 반복건수·생성 근거 표시), '상담 검색 / 플레이북' 2탭 구성(`Knowledge`) |
| 🛠 **배포 문서** | NCP·호스팅 가이드를 **앱 로그인 방식**으로 갱신(Caddy는 HTTPS+프록시만 담당) |

## 설정 (신규 환경변수)
`dashboard/.env` 에 아래를 추가합니다. **외부 서버 오픈 시 필수.**
- `APP_PASSWORD` — 대시보드 접속 비밀번호(전 직원 공용 1개). 비워두면 로그인 비활성(개발용) + 기동 시 경고.
- `SESSION_SECRET` — 세션 쿠키 서명 키(긴 무작위 문자열). 미설정 시 재시작마다 전원 재로그인.
  - 생성: `node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"`

> 블로그 검사기·AI 상담검색·플레이북 생성은 기존 `ANTHROPIC_API_KEY`(Claude)와 `CHANNELTALK_*`(채널톡)를 사용합니다.

# v1.0 → v1.2 파일별 변경 내역

추적 파일 기준 **12개 수정(+1,724 / −77)** + **신규 컴포넌트/모듈 8개**. (블로그 원본 소스 `daininc-naver-blog/`·개인설정은 제외)

## A. 신규 파일 (New)

### 1. `dashboard/server/lib/auth.mjs` (신규, 72줄)
- 로그인 인증 헬퍼. **HMAC-SHA256 서명 무상태 세션토큰**(`exp.signature`) 발급·검증, `timingSafeEqual` 기반 비밀번호/서명 비교(타이밍 공격 방지), 쿠키 파서/빌더.
- 서버 세션 저장소가 필요 없어 재시작에도 견고(`SESSION_SECRET` 고정 시). 쿠키는 `HttpOnly`+`SameSite=Lax`, HTTPS 구간에서 `Secure` 자동.

### 2. `dashboard/src/lib/auth.ts` (신규, 24줄)
- 프론트 인증 데이터 레이어: `getSession()`(로그인 필요/상태 조회), `login(password)`, `logout()`.

### 3. `dashboard/src/components/Login.tsx` (신규, 52줄)
- 로그인 화면(중앙 카드). 비밀번호 입력·오류 표시·로그인 중 상태. 성공 시 상위로 콜백.

### 4. `dashboard/src/components/BlogChecker.tsx` (신규, 243줄)
- **'블로그 검사기' 탭** 전체. 제목·본문 입력, **이미지 첨부(jpeg/png/gif/webp)**, 분석 모델 선택, 키오스크 모델 문맥 입력.
- 결과: 종합 점수·요약·핵심 키워드·제목 피드백, **문제점(심각도별)·강점**, 제목 대안, **개선 제목/본문**, 사용 모델·**비용(원) 표시**. 진행 중 취소(AbortController).

### 5. `dashboard/src/lib/blog.ts` (신규, 88줄)
- 블로그 검사 데이터 레이어: `analyzeBlog()`, `fileToImage()`(base64 변환), `BLOG_MODELS`, `SEVERITY`, `scoreColor`. 결과/이미지/입력 타입 정의(점수·이슈·비용 포함).

### 6. `dashboard/src/components/Knowledge.tsx` (신규, 25줄)
- **'꿀팁게시판' 탭 컨테이너** — `🔍 상담 검색`(`CsSearch`)과 `💡 플레이북(가이드)`(`Playbooks`)을 세그먼트 탭으로 묶음.

### 7. `dashboard/src/components/CsSearch.tsx` (신규, 191줄)
- **AI 상담 검색 화면**. 인덱스 상태 표시 + **기간(시작일~종료일) 선택 + '이 기간 수집' 버튼**(중복 제외 안내·수집 결과 요약).
- 검색창 → 원포인트 답변 카드(신뢰도 배지·조치 단계)·관련 상담 기록 목록(근거 표시·채널톡 링크).

### 8. `dashboard/src/lib/csSearch.ts` (신규, 56줄)
- 상담 검색 데이터 레이어: `getCsIndex()`, `collectCsIndex(from,to)`(기간 수집·증분), `searchCs(query)`. 인덱스 메타·검색결과·근거 타입 정의.

## B. 수정 파일 (Modified)
