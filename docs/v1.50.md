## 개요
업무현황·업무일지 화면이 정보는 많지만 **한눈에 파악하기 어렵다**는 피드백을 반영해, 두 화면의 요약·배치·상호작용을 정비했습니다.

핵심 4가지:
1. **업무현황 AI 한눈 요약** — 화면 맨 위에서 현재 업무 상황을 Claude가 요약.
2. **팀 관계도(마인드맵) 개편** — 중심 허브·가중치 협업선·업무부하/정체 표시로 정보 밀도를 높이고, 차분한 플랫 디자인으로 정리.
3. **복수 선택 + 표 필터** — 사람을 여러 명 선택하면 아래 업무표가 그 사람들 업무만 실제로 걸러서 표시.
4. **업무일지 배치 정리** — 요약을 위로, 상세를 2열로, 큰 카드는 전체폭으로 재배치.

## 새로워진 점 (요약)

| 구분 | 내용 |
|------|------|
| 🆕 **업무현황 AI 요약** | 맨 위 카드에 헤드라인(수치 포함) + 📌 지금 주목 + ⚠️ 신경 쓸 것. 업무 내용이 바뀌면 재생성, 아니면 30분 캐시. "🔄 다시 요약" 버튼 |
| ⬆️ **마인드맵** | 중심 "우리 팀" 허브 + 방사형 스포크, **협업선 굵기 = 공유 업무 수**, 요청→담당 화살표, **업무부하 링**(바쁨/과부하만) + **정체 배지**, 차분한 팔레트·플랫 디자인 |
| 🆕 **복수 선택** | 마인드맵/구성원 카드/이름칩 클릭으로 **여러 명 토글** 선택. 선택 노드·카드 강조, "선택 해제(N명)" |
| 🆕 **표 필터** | 선택된 사람(담당·협업·요청 어디든)의 업무만 **실제로 필터링**(예전엔 흐리게만). 헤더에 건수 표시 |
| ⬆️ **업무일지 배치** | 세줄요약·KPI를 맨 위 전체폭, 상세(흐름·이야기·지켜볼것·내일)는 2열 그리드, **구성원별 하루는 전체폭 그리드**로 압축 → 빈 공간 없이 한눈에 |

## 동작 메모
- AI 요약은 `ANTHROPIC_API_KEY`(.env)를 사용합니다(미설정 시 요약만 비활성, 나머지 화면은 정상).
- 요약 캐시는 업무의 (상태·정체) 조합이 바뀌면 자동 무효화됩니다.
- 마인드맵 노드 크기 = 현재 진행 중 담당 업무 수, 링 색 = 업무부하 자동 판정.

# v1.49 → v1.50 파일별 변경 내역

**수정 5개** (신규 파일 없음).

## A. 서버

### 1. `dashboard/server/notion-sales-bff.mjs`
- **`GET /api/tasks/summary`** 신규 — `loadTasks()` 집계(상태별·담당자별·정체 목록) → Claude로 한눈 요약(JSON: headline/highlights/attention) 생성. 업무 시그니처 기반 30분 캐시, `?force=1` 강제 재생성, 토큰 없으면 안내 응답.

## B. 프론트

### 2. `dashboard/src/lib/tasks.ts`
- `TaskSummary`·`TaskSummaryResponse` 타입, `fetchTaskSummary(force)` 추가.

### 3. `dashboard/src/components/TaskStatus.tsx`
- 상단 **`TaskAiSummary`** 카드(헤드라인·주목·신경쓸것, 다시요약).
- **마인드맵 재작성**: 허브·스포크·가중치 엣지(협업 곡선/요청 화살표)·부하 링·정체 배지, 플랫 디자인, 팔레트 교체.
- **복수 선택**: `selected`를 배열로 전환(`toggle`/`clearSel`, `isSel`), 마인드맵·구성원 카드·이름칩 연동.
- **업무현황표 필터**: 선택 시 관련 업무만 렌더(담당/협업/요청 매칭), 건수 표시.

### 4. `dashboard/src/components/WorkLog.tsx`
- 일간 리포트 컨테이너를 `wl-report`로, 요약(세줄+KPI)을 맨 위 전체폭, 상세를 `wl-detail`(2열)로, 구성원별 하루는 `wl-span`(전체폭)+구성원 그리드로 재배치. 작성·CS는 하단.

### 5. `dashboard/src/App.css`
- `.wl-report`/`.wl-detail`(2열 그리드·`grid-auto-flow: row dense`)/`.wl-span`(전체폭) 추가, 기존 `.brief-cols` masonry 제거, 인쇄(PDF) 규칙 갱신.
