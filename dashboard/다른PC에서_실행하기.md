# 다른 컴퓨터에서 대시보드 실행하기

지금까지 만든 **전체 현황(경영 지표) 대시보드**를 다른 PC에서 띄우는 방법입니다.
전체 현황 화면은 인터넷·서버 없이 동작하는 **샘플 데이터**라 설치만 하면 바로 보입니다.

---

## 1. 준비물 (한 번만)

- **Node.js (LTS)** 설치 — https://nodejs.org
  - 다운로드한 설치 파일을 실행하고 "다음"만 눌러 설치하면 됩니다.
  - 이미 설치돼 있으면 건너뛰세요.

## 2. 실행 (매번)

1. 압축으로 받은 `다인대시보드_배포.zip` 을 **압축 해제**합니다.
2. 폴더 안의 **`실행.bat`** 파일을 **더블클릭**합니다.
3. 검은 창이 뜨면서
   - (최초 1회만) 필요한 파일을 자동으로 내려받아 설치하고 — 몇 분 걸립니다.
   - **데이터 API 서버(BFF)** 가 새 창에서 켜지고 (`.env` 가 있을 때만),
   - **화면 서버** 가 켜지면 브라우저가 자동으로 열립니다.
4. 브라우저에 화면이 안 보이면 주소창에 **`http://localhost:5173`** 입력 후 새로고침.

> 창이 **2개**(화면 / 데이터 API) 뜹니다. 종료할 때는 두 창 모두 닫으세요.
> (화면 창에서 **Ctrl + C** → 'BFF' 창 닫기)

---

## 자주 묻는 것

**Q. "Node.js가 설치되어 있지 않습니다" 라고 나와요.**
→ 위 1번에서 Node.js를 설치한 뒤 `실행.bat` 을 다시 더블클릭하세요.

**Q. 설치가 안 되거나 멈춰요.**
→ 인터넷 연결을 확인하세요. 최초 1회 설치에만 인터넷이 필요합니다.

**Q. 다른 탭(CS 현황, 경영 지표, TR 현황 등)은 데이터가 안 나와요 / 502 오류가 나요.**
→ 그 화면들은 데이터 API 서버(BFF)와 노션·채널톡 연동(API 키 `.env`)이 필요합니다.
  `실행.bat` 은 `.env` 가 있으면 BFF를 자동으로 같이 띄웁니다.
  **보안상 배포 zip에는 `.env`(키)를 넣지 않았으므로**, 이 탭들을 쓰려면 `.env` 파일을
  별도로 전달받아 이 폴더에 넣고 다시 실행하세요.
  전체 현황 탭은 `.env` 없이도 정상 동작합니다.

**Q. 5173 포트가 이미 쓰이고 있다고 나와요.**
→ Vite가 자동으로 5174 등 다른 포트로 띄웁니다. 검은 창에 표시된 주소로 접속하세요.

---

## 수동 실행 (배치파일 대신 직접)

명령 프롬프트(또는 PowerShell)를 이 폴더에서 열고:

```
npm install     (최초 1회)
npm run dev
```
